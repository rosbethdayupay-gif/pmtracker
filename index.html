Connect your database — takes 5 minutes, free forever
This app uses Supabase (free cloud database) to store issues and photos across all devices and team members. Follow the steps below once, then it's ready.
1
Go to supabase.com and create a free account, then create a new project (pick any name like "strand-tracker"). Wait ~1 minute for it to provision.
2
In your project, go to SQL Editor and run this query to create the issues table:

CREATE TABLE issues (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at timestamptz DEFAULT now(),
  date date NOT NULL,
  property text NOT NULL,
  unit_location text NOT NULL,
  description text NOT NULL,
  category text NOT NULL,
  status text NOT NULL DEFAULT 'Open',
  inspector text,
  photos text[] DEFAULT '{}',
  addressed_at timestamptz,
  addressed_by text,
  comments jsonb DEFAULT '[]'
);

ALTER TABLE issues DISABLE ROW LEVEL SECURITY;

INSERT INTO storage.buckets (id, name, public) VALUES ('issue-photos', 'issue-photos', true);

CREATE POLICY "Public Access" ON storage.objects FOR ALL USING (bucket_id = 'issue-photos') WITH CHECK (bucket_id = 'issue-photos');
3
Go to Project Settings → API. Copy your Project URL and anon public key, and paste them below.
